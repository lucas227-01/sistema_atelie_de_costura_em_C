# sistema_atelie_de_costura_em_C
Sistema modular em C para gestão de ateliês de costura. Permite o cadastro de serviços (CRUD) e o lançamento de ordens de serviço com múltiplos itens por cliente. Desenvolvido com persistência de dados em arquivos TXT, tratamento robusto de buffer de teclado e suporte a nomes com espaços.
